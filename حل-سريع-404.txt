السبب:
❌ التطبيق غير مشغّل أو المسار خاطئ

═══════════════════════════════════════════════════════
   ✅ الحل (5 خطوات)
═══════════════════════════════════════════════════════

1️⃣ سجل دخول cPanel
   https://hshestate.au:2083

2️⃣ افتح "Setup Node.js App"

3️⃣ تحقق من:
   □ هل يوجد تطبيق؟
   □ هل الحالة "Running"؟
   
   إذا لا → اتبع الخطوات أدناه

4️⃣ افتح File Manager
   □ اذهب إلى: public_html/table/
   □ هل ترى: server.js, package.json, public/?
   
   نعم → انسخ المسار الكامل
   
   مثال: /home/hshestat/public_html/table/voice-chat-room
         أو
         /home/hshestat/public_html/table

5️⃣ أنشئ تطبيق جديد
   في Setup Node.js App → Create Application:
   
   Node.js version: 18.x
   Application mode: Production
   Application root: [المسار الذي نسخته من الخطوة 4]
   Application URL: hshestate.au
   Application startup file: server.js
   
   → Create
   → Run NPM Install
   → Start App
